# Face-Recognition-Attendance-System

### Features:

1. **Home Page**
   - **Student Management System**
     - Save
     - Take Photo Samples
     - Update
     - Delete
     - Clear
   - **Train Photo Samples**
   - **Take Attendance with Face Detection**
   - **Attendance Report**
     - Excel File
     - SQLite Database
   - **Developer Page**
   - Help Desk

### Technologies Used:

- **Python**
- **OpenCV**
- **Tkinter GUI**
- **SQLite Database**

This project leverages real-time face detection to streamline student attendance management.
